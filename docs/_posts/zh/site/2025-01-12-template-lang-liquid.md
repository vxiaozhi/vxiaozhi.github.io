# 模板编程语言-Liquid

Liquid 是一门开源的模板语言，由 Shopify 创造并用 Ruby 实现。它是 Shopify 主题的骨骼，并且被用于加载店铺系统的动态内容。

从 2006 年起，Liquid 就被 Shopify 所使用，现在更是被大量 web 应用所使用。

jekyll 采用 Liquid 作为器默认模板语言。

- [github](https://github.com/Shopify/liquid)
