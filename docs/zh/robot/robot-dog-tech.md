# 机器狗技术

国内外所有搞机器狗的公司，全都是基于 MIT 的开源项目基础上做的。包括 机器人操作系统 ROS 和 Mini Cheetah。

## ROS 

整个ROS的项目在部署运行过程中，分为几个部分：

环境
执行器
传感器
软件结构
